## How to Use
1. Edit `index.html` and `style.css` to personalize content, branding, and contact details.
2. Host for free using GitHub Pages (see below).

## Free Hosting on GitHub Pages
1. Create a free GitHub account if you don't have one.
2. Create a new repository named `curehunt.github.io` (or your preferred username).
3. Upload your website files (`index.html`, `style.css`, etc.) to the repository.
4. Go to the repository's Settings > Pages and set the source branch (usually `main`).
5. Your site will be live at `https://curehunt.github.io`.
